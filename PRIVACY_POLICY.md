# Политика конфиденциальности для расширения AnimeGO+

## 1. Сбор данных
Расширение **AnimeGO+** не собирает, не хранит и не передает на сторонние серверы ваши личные или пользовательские данные. Вся работа расширения происходит исключительно локально в вашем браузере.

## 2. Использование разрешений (Permissions)
Расширение запрашивает следующие права доступа для корректной работы:
*   **`storage`**: используется для сохранения ваших локальных настроек плеера (например, авто-пропуск опенингов, включенные модули). Данные никогда не покидают ваше устройство.
*   **`activeTab` и `host_permissions`** (аниме-сайты и плееры, такие как animego.me, kodik.info и др.): используется для получения доступа к воспроизводимому видео, чтобы добавить поверх него кнопки управления (пропуск интро, автовоспроизведение) и улучшить интерфейс.

## 3. Передача данных
Мы не продаем, не обмениваем и не передаем вашу информацию третьим лицам. Расширение не содержит средств аналитики или отслеживания пользователей.

## 4. Изменения в политике конфиденциальности
Мы оставляем за собой право вносить изменения в эту политику конфиденциальности. Любые изменения будут отражены на этой странице.

## 5. Контакты
Если у вас есть вопросы по политике конфиденциальности, вы можете открыть Issue в репозитории расширения на GitHub или оставить отзыв на странице расширения.
