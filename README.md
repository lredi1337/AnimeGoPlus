📺 AnimeGO+ – Расширение для браузера, которое делает просмотр аниме на AnimeGO максимально комфортным, избавляя от лишних кликов и расширяя возможности плеера.

✨ Основные фишки: 🚀 Марафон: Автоматическое нажатие «Play» и бесшовный переход к следующей серии в конце эпизода.

⏩ Smart Skip: Автоматический или ручной пропуск опенингов и эндингов (поддерживает сцены после титров!) благодаря интеграции с AniSkip API.

📱 Умное всплывающее меню (Popup): Глобальный выключатель расширения и встроенный Поиск Музыки. Узнайте название опенинга/эндинга из текущей серии и мгновенно найдите трек на YouTube в один клик!

📊 Быстрые ссылки: Прямые интерактивные кнопки перехода на профиль текущего аниме в Shikimori и MyAnimeList (доступны как в блоке рейтинга на сайте, так и в меню расширения).

🔄 Двусторонняя интеграция: Умная кнопка «▶ Смотреть на AnimeGO», которая автоматически появляется под постерами прямо на страницах аниме на сайтах **Shikimori** и **MyAnimeList**. Один клик — и вы смотрите выбранный тайтл на AnimeGO!

🖥️ Pseudo Fullscreen: Кастомный режим «плеер на всё окно браузера», который работает корректно даже при автоматическом переключении серий и убирает все визуальные рамки.

⌨️ Продвинутые Hotkeys: Полностью настраиваемые горячие клавиши для управления громкостью (Mute), перемоткой, переключением серий и фуллскрином.

🖱️ Интерактивный плеер: Двойные клики по краям для перемотки, визуальные зоны опенингов на таймлайне и всплывающие уведомления (Flash-иконки) при действиях.

🛡️ Оптимизация и безопасность: Минимальная нагрузка на систему благодаря событийно-ориентированной архитектуре (MutationObserver) и встроенная проверка безопасности данных между окнами.

⚙️ Меню настроек: Удобный интерфейс для кастомизации всех функций, доступный по клику на шестеренку прямо над плеером или через меню расширения.

📦 Как установить:
1. Скачай этот репозиторий (кнопка Code -> Download ZIP).
2. Распакуй архив в удобную папку.
3. Открой Chrome и перейди по адресу chrome://extensions/.
4. Включи «Режим разработчика» (переключатель справа сверху).
5. Нажми «Загрузить распакованное расширение» и выбери папку с проектом.
